DEFINICIONES:

API: Interfaz de programación de aplicaciones ofrecida por Círculo de Crédito a través de su Portal API HUB, que le permite el acceso y uso del contenido y/o servicios API.

Contenido API: Son aquellos datos o información disponible a través de un API.

Servicios API: Son las funcionalidades accesibles o ejecutadas a través de un API concreta, de acuerdo con los términos y condiciones establecidos en el presente Contrato. Un API provee al menos una funcionalidad.

Aplicación: Sitio de internet, aplicación para teléfonos móviles, tableas y/o una aplicación informática diseñada para sus clientes, empresas o cualquier persona que acceda o utilice el Contenido API.

Cliente/Empresa/Usted/Usuario: Se refiere a la persona que utiliza este Portal API HUB, quien cuenta con las facultades suficientes y capacidad legal para obligarse en los términos y condiciones que se indican en este Contrato.

Cookie: archivos de texto que contienen pequeñas cantidades de información que se descargan en un dispositivo en el momento de visitar algún sitio de Internet

Datos/entorno de simulación SANDBOX: Se refiere a un ambiente de pruebas que contiene datos falsos, simulados, controlados utilizados para realizar cualquier tipo de prueba en el API y que únicamente serán utilizados para este fin.

Documentos: Son aquellos manuales, tutoriales, descripciones sobre el funcionamiento de las APIS y/o servicios API, guías de configuración, seguridad, conectividad, así como material de apoyo y/o ayuda que sea utilizado como referencia para explicar el uso de las APIS y/o servicios API publicados en el portal de API HUB.

Información Confidencial: Aquella Información, documentos, credenciales, “Tokens”, contraseña de un solo uso (One-Time Password “OTP”), entregados por Círculo de Crédito a Usted para el uso y acceso del Contenido y Servicios API y el portal API HUB.

Metadatos: Se refiere a los datos que describen el Contenido API y/o los servicios API.

Portal API HUB: Sitio de internet a través de la URL: http://developer.circulodecredito.com.mx
